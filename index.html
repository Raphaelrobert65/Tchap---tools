Outil interne Tchap

Tchap - Tools

Outil interne pour retrouver une instance Tchap à partir d’un domaine, générer un MXID à partir d’une adresse email professionnelle et vérifier rapidement le statut d’ouverture d’un domaine.

3 fonctionnalités · même logique · même charte
Fonctionnalité 1

Retrouver une instance

Saisis un domaine professionnel pour identifier automatiquement l’instance Tchap associée.

Fonctionnalité 2

Générer un MXID

Saisis un email professionnel. L’outil retrouve l’instance et construit automatiquement le MXID.

Fonctionnalité 3

Vérifier si un domaine est ouvert

Vérifie rapidement le statut métier d’un domaine à partir de l’instance Tchap retournée.

Ouvert sur Tchap si une instance dédiée est trouvée · non ouvert si le domaine bascule sur externe.
