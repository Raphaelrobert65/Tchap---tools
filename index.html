Tchap - Tools

Outil interne pour retrouver une instance Tchap à partir d’un domaine et générer un MXID à partir d’une adresse email professionnelle.

Fonctionnalité 1

Retrouver une instance

Saisis un domaine professionnel pour identifier automatiquement l’instance Tchap associée.

Fonctionnalité 2

Générer un MXID

Saisis un email professionnel. L’outil retrouve l’instance et construit automatiquement le MXID.

Fonctionnalité 3

Vérifier si un domaine est ouvert

Saisis un domaine. Si l’instance retournée est externe, le domaine est considéré comme non ouvert. Sinon, il est considéré comme ouvert.

Règle métier appliquée : instance dédiée = domaine ouvert · instance externe = domaine non ouvert.
